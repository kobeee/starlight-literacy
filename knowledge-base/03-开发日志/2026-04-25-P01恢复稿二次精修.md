## 视觉判断

### 1. 通关圆球确实偏冷

原通关态主色：

- 主球：`#B8B0B0`
- 阴影：`#8A8580`

问题：这组颜色更像水泥灰，在暖米底、蜂蜜金、田园水彩里显脏且偏冷；尤其和当前关大金球相邻时，会产生“旧 UI 贴纸”的割裂感。

### 2. 单屏左侧白色不是独立节点

检查 `IF6la` 后确认：

- 单屏预览由 `SsbLg` 长卷副本 `y=-460` 取景形成。
- 左侧白色主要来自背景图最左边缘的亮云 / 高光留白，不是按钮、遮罩或误插入的白色节点。

处理方式：不整体移动当前关节点，只调整单屏副本里的底图与暖米滤镜横向取景，让背景边缘吃进插画主体，减少硬白边感。

## 本次改动

### 通关态暖化

长卷与单屏副本同步处理 `u1/u2/u3`：

- 通关主球先改暖石子色后复核仍偏灰，最终改为蜂蜜饼干色 `#E1B46A`
- 通关球内描边改为奶油黄 `#FFF0C8`
- 勾选 icon 改为奶白 `#FFF8E8`
- 阴影改为轻暖棕 `#9C6A2F4D`
- 标签正文保持暖深棕 `#5D4A36`
- 副文案改为暖灰棕 `#8B7A66`

效果：仍然能和当前关金球拉开层级，但不再像灰色 UI 贴片。

### 未解锁态与雾带暖化

远方锁态同步从冷灰改为雾里的浅米褐：

- 未解锁球先改浅米褐后复核仍偏灰，最终改为浅麦色 `#EBD8A8`
- 锁 icon：`#B58A4A`
- 远方标签：`#8A6B3D`
- 远方副文案：`#A88D64`
- 顶部雾带从纯白雾改成暖米雾：
  - `#FFF8EFCF`
  - `#FFF4E0A0`
  - `#FFF8EC50`
  - `#FFF8EC00`

效果：保留“远方规模感”，但更像暖阳晨雾，不是白色遮罩糊上去。

### 当前关指引徽章降警示感

原指引徽章红色较接近警示色。本轮改为暖珊瑚：

- `#E85D5D` → `#F06F5B`

效果：仍然能在 200ms 内引导孩子点击，但不再像错误/警告提示。

### 路径点降噪

原路径点黑棕较重，局部像杂点；复核时也会和关卡圆一起被读成“灰圆”。本轮统一改为：

- fill `#D4A14A99`
- stroke `#FFF4DA`

效果：路径节奏还在，但更轻、更暖，不再增加灰感。

### 关卡圆去灰复核

2026-04-25 追加复核后确认：红框关注的关卡圆“仍灰”主要来自三处：

- 已通关 `u1/u2/u3` 的主圆仍偏石灰色
- 未解锁 `u6/u7/u8` 在雾里仍读成灰球
- 路径小圆点偏灰褐，会强化整条路线的灰感

已在 `9wPMd` 与 `IF6la` 副本同步去灰。当前仍能看到的路上空白椭圆台阶属于底图插画内容，不是独立 Pencil 节点；本轮未重绘底图。

### 单屏左边缘处理

仅在 `IF6la` 的长卷副本内处理背景层：

- 复核时发现 `x=-18`, `width=426` 会把背景图左缘的浅绿/白色竖带带进视窗，红框内并未完全生效。
- 已改回与长卷本体一致的取景：`rXgrb` / `qulAl` 均为 `x=0`, `width=390`。

说明：`IF6la` 仍通过父级 `SsbLg y=-460` 取当前关居中视窗；`snapshot_layout` 只应提示 `SsbLg` 被父级裁切，这是预期取景，不再有背景/滤镜自身外扩裁切。
