## 踩坑记录

1. **小旗方案失败**：原计划在球右上角加 `GO!` 红旗 + 旗杆。
   - rotation:-8 + frame fit_content 组合 → frame 渲染空白（fully clipped）
   - 改 fixed-width 去掉 rotation → 仍空白
   - **根因推测**：I() 从零建带旋转的复杂 frame 仍会触发"静默失败"，与 04-19 记录的 I() 复杂 frame 渲染失败同源
   - **解法**：放弃旗子，改用 ellipse + icon_font 的"指引徽章"，更直白

2. **新建 ellipse 也失败**：直接 I() 建一个 44×44 红 ellipse + icon → 截图空白
   - **解法**：用 `C("9rvr7", ...)` 复制已工作的 u1 球节点，再 U() 改色和图标名 → 一次成功
   - **再次印证**：复杂场景中 **C() 复制 > I() 新建**，是 Pencil 的硬规律，不是个例

3. **K2tsy 是 P01-v2 的独立副本，不是 ref**：所有改动必须在 pi991 和 K2tsy 上各做一次，descendants 子节点 ID 不同。
