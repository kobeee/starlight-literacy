# 2026-04-23 P04/P05/P08/P09/P11 田园暖彩迁移

## 背景

P02/P03 已确认“统一调性不是单一色调”：以 `#FAF6F0` 奶油米为基底，蜂蜜金 `#FFC947` 作为主强调，暖深棕 `#5D4A36` 做主文字，同时允许低饱和花田色板作为局部信息区分。

本轮继续把旧视觉中残留的粉橙渐变、紫色 step、夜空蓝底、粉色付费 CTA 和廉价 glow 收敛到同一套暖阳田园滤镜。

## 落地范围

### P04 书写页

- 背景从粉色渐变改为 `#FAF6F0` 奶油米。
- 田字格从粉色描边/glow 改为蜂蜜金边框 + 暖米纸面。
- 描红字从粉红改为低饱和浅金棕，降低“错误红”的心理暗示。
- step bar 当前态统一为蜂蜜金，历史态统一为暖灰。
- 紫色撤销按钮、粉色 glow 和多余彩色装饰弱化或删除。

### P05 / P06 练习页

- 主题卡从粉橙多彩渐变改为 `#FFEDB8` 浅金米黄场景卡。
- 大字改为单色金 `#E8A800`，不再用白字+重阴影。
- 选项卡统一为暖白底，选中态用蜂蜜金描边和轻阴影。
- 反馈条统一为金色正反馈，并补齐“答对拿 3 颗星 · 错了也有星星～”文案。
- 顶部 step bar 去掉薄荷/天蓝/粉红平均分配，改成暖灰历史态 + 蜂蜜金当前态。

### P08 测验结果

- 从深蓝夜空结果页迁移为暖日庆祝页。
- 保留庆祝插画，但降低不透明度并放到背景层，避免压住标题和数据。
- 标题从白金渐变改为暖棕，提升浅底可读性。
- 数据卡从半透明蓝底改为暖白底 + 金边，并保留金/绿/蓝三类数据区分。
- 主按钮从粉色渐变改为蜂蜜金 CTA，分享按钮改为暖白次按钮。
- 删除底部被裁剪的奖章回声装饰。

### P09 付费引导

- 背景改为奶油米，去掉桃橙渐变。
- 单元包选中卡改为暖白底 + 蜂蜜金边，保留价格信息层级。
- CTA 从珊瑚粉改为蜂蜜金，文字改暖棕，降低“促销感”。
- Glow 和粉色装饰弱化为金/棕低透明装饰。

### P11 家长中心

- 背景从紫灰渐变改为奶油米。
- 卡片改为暖白底 + 暖灰描边，保持家长端可信、安静。
- 今日统计保留三色信息区分，但统一为低饱和暖彩。
- 累计进度条从粉橙渐变改为蜂蜜金单色。
- 设置项从彩色碎片改为暖黄/暖灰/薄荷三段，图标和文字统一暖棕体系。
