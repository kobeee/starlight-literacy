# 2026-04-23 P02 / P03 田园暖彩调性统一

## 背景

上一轮（2026-04-22）反 AI 迁移把 P02/P03 的"紫"换成了"深棕"（#3A2D20 大色块 + #3D3D3D 近黑文字），矫枉过正——
P02 banner 像咖啡店菜单，P03 "下一步"按钮压抑死沉，跟 P01 田园彩虹长卷的氛围严重脱节。

用户原话："色调有点深褐，跟产品理念不吻合，首页地图调性挺好，整个产品要统一色调。"

## 概念纠偏：单色调 ≠ 统一色调

**第一版改错了方向**——把所有深棕换成蜂蜜金 + 暖棕 + 奶油米三件套，结果干净但寡淡。

用户纠正："色调，不是单一色调。能理解不？"

**正确理解**：
- P01 之所以好看不是因为颜色少，而是**所有颜色都被一层"暖阳田园滤镜"罩住**——彩虹、花海、柯基、蓝天，色彩丰富但和谐。
- "统一调性"=共享同一个色彩温度（低饱和、暖光晒过），而不是减颜色数量。

## 落地方案：田园暖彩色板

|---|---|---|
| 草绿 | `#D8E8C5` 底 / `#9CC077` 边 / `#3D5C20` `#5D6F45` 字 | 一/上/月/水…（自然系） |
| 樱粉 | `#FFD9D9` 底 / `#F5A8A8` 边 / `#8B2D2D` `#8B5050` 字 | 二/人/火/目…（生命系） |
| 天蓝 | `#D4E8F0` 底 / `#8FBED4` 边 / `#1F4858` `#3D6878` 字 | 三/口/水/天…（清凉系） |
| 橘黄 | `#FFE0C2` 底 / `#FFB87A` 边 / `#9C5418` `#8C5A30` 字 | 大/手/木…（温暖系） |
| 蜂蜜金 | `#FFE9B8` 底 / `#FFC847` 边 / `#8B5E00` `#8B6A2D` 字 | 小/日/土…（阳光系） |

**基底**：奶油米 `#FAF6F0`（不变）
**主角色**：蜂蜜金 `#FFC947`（CTA / 当前态徽章 / banner 边框）
**主文本**：暖深棕 `#5D4A36`（替代 #3D3D3D / #3A2D20）

## P02 单元入口改造

- banner：深棕 `#3A2D20` → `#FFEDB8` 米黄底 + `#FFC947` 金边 + 插画放大 120→170 露脸主角
- 20 字卡：从单色 `#F4EDDF` → 5 色交替花田（草绿/樱粉/天蓝/橘黄/蜂蜜金）
- CTA "开始学习"：深棕 → 蜂蜜金底 + 暖棕字 + 金色发光阴影 `#FFC94755`
- 文字色：`#3D3D3D` → 对应字卡色系的低饱和暖色

## P03 认字页改造

- "下一步"按钮：深棕 → 蜂蜜金 + 暖棕字
- step bar 当前态："认"徽章 `#3A2D20` → `#FFC947` + 暖棕字/icon
- 组词卡：3 张白底 → 橘/蓝/粉三色，对应字色用对应色系深棕（火焰`#9C5418` / 火车`#1F4858` / 大火`#8B2D2D`）
- 卡片描边：`#EADFCB` → `#FFC947` 金边

## 工具笔记

1. **批量改字卡用 U() 一次性发**：26 个 cell × 2 属性（fill + textColor）一次 batch_design 搞定，没卡。
2. **export_nodes 报错**："failed to execute tool call. you are probably referencing the wrong .pen file"——原因待查，绕开方案是单页 get_screenshot。
3. **absolute-in-vertical 老 bug 复现**：往 banner 里 I() 装饰星星/太阳没渲染（已记忆），banner 本身已够趣味，没强求。

## 决策记录

- **不再追求"色彩极简"**：儿童产品 vs 成人极简产品的根本差异——孩子需要色彩刺激，但要"被驯服的色彩"（低饱和暖光），不是数码原色。
- **首页 = 调性基准**：今后所有页面的色彩选择都对照 P01 调色盘判断"是否在同一暖阳滤镜下"。
- **暂未推 P04/P05/P08/P09/P11**：等用户确认 P02/P03 调性后，新 session 继续按此色板批量迁移。
