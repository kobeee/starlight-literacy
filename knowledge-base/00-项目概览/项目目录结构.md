## 前后端分离说明

- **前端** (`src/miniprogram/`)：微信小程序原生开发 + Skyline 渲染引擎
- **后端** (`src/cloud/`)：微信云开发（Serverless），云函数处理业务逻辑
- 两者通过微信云开发 API 通信，无需自建服务器和域名备案

## 多端扩展说明

当前聚焦微信小程序端。未来视用户反馈可扩展 iOS / Android。

- **跨端共享**：`assets/`（素材）和 `data/`（内容数据）是真正的共享资产
- **各端独立**：UI 代码各端独立实现，AI 辅助重写成本低
- **知识库即文档**：`knowledge-base/` 中的系统设计文档是最有价值的"复用"，做新端时看文档即可快速重建
