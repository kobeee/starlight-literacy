## Design Context

### Users
3~6 岁学龄前儿童（主要操作者）+ 家长（付费决策者、旁观辅导者）。
孩子在家庭环境中使用，注意力短暂（3~5 分钟/轮），依赖图形、颜色和声音理解界面。
核心任务：通过“玩 → 认 → 练 → 写 → 小测/鼓励”这条轻闭环，让孩子在没有挫败感的前提下反复接触汉字。

### Brand Personality
**天真 · 无邪 · 快乐 · 幸福 · 无忧无虑 · 有趣 · 好玩**
(Innocent · Pure · Joyful · Happy · Carefree · Playful)

情绪目标：
- 孩子感到：**轻松、好奇、被鼓励、愿意继续点，像在晒太阳的绘本里玩**
- 家长感到：**安心、信赖、温暖、有审美，不像廉价 AI 模板**

品牌声音：像一位温柔又有点俏皮的幼儿园老师，永远鼓励，不催促，不责备。

### Aesthetic Direction
**视觉基调**：田园暖彩 + 晒暖绘本 + 低饱和童趣

- 参考方向：洪恩识字的场景化、绘本书页感、晒暖的儿童插画、自然系生活方式视觉
- 反参考：紫蓝梦幻渐变、暗底星空、霓虹 glow、糖果六色平均分配、满屏贴纸感装饰
- 核心意象：**阳光、花田、小屋、云朵、小动物、蜂蜜金、暖米纸**
- 页面目标：每一屏都像“一个小世界”，但这个世界发生在同一片暖阳里

### Color Strategy
**主结构**：
- 大面积中性色：奶油米 `#FAF6F0`、暖白 `#FFFAEC`
- 单一主强调：蜂蜜金 `#FFC947`
- 主文字：暖深棕 `#5D4A36`

**辅助分组色**：
- 草绿 `#D8E8C5 / #9CC077`
- 樱粉 `#FFD9D9 / #F5A8A8`
- 天蓝 `#D4E8F0 / #8FBED4`
- 橘黄 `#FFE0C2 / #FFB87A`
- 薄荷 `#F0FAF6 / #5BA88E`

**规则**：
- 一个页面最多一个主强调色
- 辅助色只做信息分组或局部趣味，不能抢主视觉
- 禁止把“丰富”理解成“高饱和 + 多渐变”

### Illustration Strategy
- 每页可以有一个主插画区，但插画服务于情绪，不抢任务
- 优先自然感、生活感、故事感，而不是魔法粒子和特效
- 星星元素允许保留，但它只是品牌痕迹，不是视觉引擎
- 目标审美：**家长愿意截图，孩子愿意继续点**

### Design Principles

1. **一屏一件事**
   孩子不需要复杂导航，只需要知道现在点哪里最开心。

2. **暖光统一，不是单色统一**
   页面可以有辅助色，但都要像被同一层太阳光晒过。

3. **厚实可按**
   按钮、字卡、标签都要像能摸到的东西，而不是轻飘飘的 UI 贴图。

4. **快乐来自鼓励，不来自轰炸**
   小奖励、小星星、小庆祝可以有，但不能靠 glow 和彩色爆炸堆情绪。

5. **孩子先舒服，家长再认可**
   孩子先觉得轻松好玩，家长再感到高级、有效、值得付费。

6. **绘本感优先于界面感**
   让页面像故事书的一页，而不是企业后台换成儿童插画。

### Implementation Notes
- Pencil 阶段可先用系统字体保结构，开发阶段再引入更有童趣的品牌字体
- 避免把旧“梦幻渐变/星空童话”词汇重新写回任何高权重文档
- 视觉判断以当前 `design.pen` 中已完成的暖彩页面为准，而不是早期实验稿
